# ShopSmart
Smart Product Review &amp; Affiliate Website
ShopSmart is a modern, responsive product showcase website designed to display curated products with pricing, interactive search functionality, and call-to-action purchase buttons.

This project demonstrates strong frontend development skills using HTML, CSS, and JavaScript, with a clean structure suitable for affiliate marketing integration and future scalability.


✅ Clean and modern UI design

✅ Responsive layout

✅ Product listing cards

✅ Live product search filter

✅ Buy button with redirection functionality

✅ Notification popup system

✅ Organized and maintainable code structure

✅ Separate HTML, CSS, and JavaScript files
